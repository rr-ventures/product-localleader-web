# Local Leader

A product landing page for a local trades marketing service — "Become the #1 Tradie in Your Area."

## What It Is

A polished Astro marketing site with a full landing page layout: hero, testimonials, solutions, services, pricing, features, FAQ, founder block, and CTA. Designed for lead generation and conversion.

## Sections

- Hero with headline and primary CTA
- Testimonials and social proof
- Solutions overview
- Services breakdown
- Pricing tiers
- Brand logos
- Features grid
- FAQ
- Founder block
- Final CTA

## Stack

- Astro 4
- Tailwind CSS
- TypeScript

## Quick Start

```bash
npm install
npm run dev
```
