Real Estate Seller Lead - Ad Creative Templates

Overview: Use this document to access the image ad creative templates we use for our real estate seller lead generation campaigns on Facebook & instagram. 

Rules for making creatives: 
Always make sure you incorporate the clients branding guidelines. This should have been confirmed in the onboarding call & recorded in the client file in clickup. 
When incorporating the clients branding guidelines consider the following elements can be included in the creative templates. Branding colours, fronts, logos, existing images etc. 
If the client does not have a preference for branding colours then use red as the primary colour in the ad graphics. COLOUR CODE = #FF1616
Always make sure that text is readable and that colours are selected so that there is sufficient contrast between image text & graphics.
