# Local SEO Fulfillment SOP

Based on Caleb Ulku's Core 30 framework + GBP optimization system.
This is your step-by-step delivery engine for every client.

---

## Phase 0: Client Onboarding (Day 0)

### Information Gathering
- [ ] Business name (exact legal name — must be consistent EVERYWHERE)
- [ ] Physical address (or service area if no storefront)
- [ ] Phone number (the ONE number that will be used everywhere)
- [ ] Website URL (or note if they need one built)
- [ ] Google Business Profile login credentials (or set up Zoom call to get access)
- [ ] List of ALL services they offer (aim for 20-30)
- [ ] Top 3 competitors they're competing with for market share
- [ ] Their existing customer database/contact list (for review campaign)
- [ ] Any existing branding assets (logo, photos, colors)
- [ ] Target suburbs/areas they service

### Baseline Measurements (BEFORE you touch anything)
- [ ] Screenshot current Google organic rankings for "[service] [city]"
- [ ] Screenshot current Google Maps position for main keywords
- [ ] Set up LeadSnap rank map — record baseline top-3% coverage
- [ ] Screenshot Google Business Profile insights (calls, views, searches)
- [ ] Note current review count and average star rating
- [ ] Run Screaming Frog crawl on existing site (if they have one) — export title tags, H1s, word counts
- [ ] Check existing citations with BrightLocal or Moz Local for NAP consistency

> **Why baseline matters:** This is how you PROVE results later. No baseline = no case study. Do this before changing a single thing.

## Phase 1: Google Business Profile Optimization (Days 1-2)

### Categories
- [ ] Set primary category to their main revenue-driving service
- [ ] Add 2-3 secondary categories that match services they actually provide
- [ ] Check top 3 competitors' categories using GMB Everywhere chrome extension
- [ ] Every secondary category MUST have a matching page on the website

### Services List
- [ ] Expand to 20-30 services listed in GBP
- [ ] Each service needs: name, description (150-300 characters)
- [ ] Service names must EXACTLY match what the website pages will say
- [ ] Group services under the correct categories

### Business Description
- [ ] Write full 750-character description
- [ ] Include: primary category, city name, 2-3 secondary services, years in business, areas served
- [ ] Natural language — NOT keyword stuffed

### Photos
- [ ] Upload 10+ photos: team, job sites (before/after), equipment, exterior
- [ ] Geotag all photos with GPS coordinates matching service area (use GeoImgr)
- [ ] Add photo descriptions with primary category + city where natural
- [ ] Plan for weekly photo uploads going forward

### Q&A Section
- [ ] Seed 5-10 Q&As covering: service areas, pricing, response times, qualifications, process
- [ ] Answer each one from the business account
- [ ] These get crawled by Google and can appear in AI Overviews

### GBP Posts
- [ ] Create first post (service spotlight or recent project)
- [ ] Set up weekly posting schedule (minimum 1/week)
- [ ] Post types: service spotlights, before/after, seasonal offers, community involvement

### GBP Embed
- [ ] Embed Google Maps/GBP on homepage using embed code
- [ ] Place visibly on homepage, NOT buried in footer

## Phase 2: Website Structure — Core 30 Build (Days 3-7)

### The Exact-Match Rule (MOST IMPORTANT)
The language on the website MUST exactly match the language in the GBP. Not synonyms. Not variations. Exact match.
- GBP primary category = "Painter" → Homepage title = "Painter in [City]"
- GBP service = "Interior Painting" → Service page title = "Interior Painting in [City]"

### Homepage Restructure
- [ ] **Title tag:** `[Primary Category] [City Name]` (e.g., "Painter Gold Coast")
- [ ] **H1 tag:** Same formula as title tag
- [ ] **First paragraph:** 50-100 words with primary category, city, years serving, 2-3 service mentions, areas served
- [ ] **H2 sections:** One for each secondary GBP category, with 50-100 word blurb + link to category page
- [ ] **GBP embed:** Google Maps embedded visibly
- [ ] **Click-to-call:** Phone number visible without scrolling on mobile
- [ ] **Schema:** LocalBusiness schema with name, address, phone, geo coordinates, openingHours, serviceArea

### Category Pages (1 per GBP secondary category)
- [ ] **Title tag:** `[Secondary Category] [City]`
- [ ] **H1:** Same as title tag
- [ ] **Opening paragraph:** 100-150 words establishing authority in this service + city
- [ ] **H2 subheadings:** One per service under this category
- [ ] **50-100 words** under each H2 linking to individual service page
- [ ] **Local FAQ section:** 5+ questions about this service category
- [ ] **Schema:** Service schema + LocalBusiness schema
- [ ] **Minimum 1,000 words total**

### Service Pages (1 per GBP service — build most important ones first)
- [ ] **Title tag:** `[Service Name] in [City]`
- [ ] **H1:** Same or close variation
- [ ] **Content:** 800-1,200+ words
- [ ] **Local flavor:** specific neighborhoods, local weather/conditions relevant to service, local landmarks
- [ ] **Process section:** What happens when they call / what the service involves
- [ ] **Benefits section:** Why hire a professional, risks of DIY
- [ ] **FAQ section:** 5+ questions specific to this service
- [ ] **Testimonials/reviews section**
- [ ] **Clear CTA:** Phone number + contact form
- [ ] **Schema:** Service schema

### Internal Linking (The Wheel Method)
- [ ] Homepage → links to all Category Pages
- [ ] Category Pages → link to their Service Pages + back to Homepage
- [ ] Service Pages → link back to Category Page + Homepage
- [ ] Cross-link related services where natural
- [ ] All pages accessible within 2-3 clicks of homepage

### Anti-Slop Content Checklist (for AI-generated content)
Every page must include at least 2-3 of these LOCAL signals:
- [ ] Specific neighborhood/suburb mentions (not just the city)
- [ ] Local weather/climate and how it affects the service
- [ ] Local building codes or permits if relevant
- [ ] Local landmarks as proximity references
- [ ] Seasonal relevance to local climate
- [ ] Reference to specific local areas they service

> **Test:** Could this page be about ANY city with just the name swapped? If yes, it's slop. Rewrite.

## Phase 3: Technical SEO (Days 7-8)

- [ ] Run Google PageSpeed Insights — target sub-3 second mobile load
- [ ] Core Web Vitals: LCP < 2.5s, CLS < 0.1, FID < 100ms
- [ ] Compress all images (TinyPNG or similar)
- [ ] Submit sitemap to Google Search Console
- [ ] Check robots.txt isn't blocking important pages
- [ ] Run Screaming Frog: verify all title tags, H1s, no broken internal links, no redirect chains
- [ ] All pages mobile-friendly and readable
- [ ] Fix any 302 redirects → convert to 301s
- [ ] Verify all Core 30 pages are within 2-3 clicks of homepage

---

## Phase 4: Citations & External Validation (Days 8-10)

### Super Citations (Tier 1 — Do these first)
- [ ] Google Maps (already done via GBP)
- [ ] Bing Places for Business (verified)
- [ ] Apple Maps via Apple Business Connect (verified)
- [ ] Facebook Business Page (claimed + completed)
- [ ] Yelp (verified + claimed)
- [ ] Yellow Pages (AU/NZ equivalent)
- [ ] True Local (AU)
- [ ] Hotfrog (AU)

### Standard Citations (Tier 2)
- [ ] Submit to 50+ platforms via LeadSnap
- [ ] Industry-specific directories (e.g., HiPages for trades in AU)
- [ ] Local business directories
- [ ] Verify NAP is BYTE-FOR-BYTE identical across every listing

### NAP Consistency Audit
- [ ] Same business name everywhere (including "Pty Ltd" or "LLC" — pick one format)
- [ ] Same address format everywhere (e.g., "Unit 4" not "U4" on some and "Unit 4" on others)
- [ ] Same phone number format everywhere

## Phase 5: Reputation Management Setup (Days 10-12)

### GoHighLevel Sub-Account
- [ ] Create GHL sub-account for client
- [ ] Connect to their Google Business Profile
- [ ] Upload existing customer database/contact list

### Review Campaign
- [ ] Set up automated review request workflow
- [ ] Template message: personal, includes direct link to Google review
- [ ] Optional: incentive (e.g., "leave a review to go in the draw for a $200 Bunnings voucher")
- [ ] Send first batch of review requests
- [ ] Monitor and respond to ALL reviews (including old unanswered ones)

### Ongoing Review Management
- [ ] Weekly check on new reviews
- [ ] Respond to every review (positive and negative)
- [ ] Target: 4.5-4.8 star average (the sweet spot — below 4.5 hurts trust, obsessing over 5.0 is vanity)

## Phase 6: Local Link Building (Month 1-2, Ongoing)

### First Link (Highest Priority)
- [ ] Local Chamber of Commerce membership ($150-400/year) — easiest high-value local link

### Community Links
- [ ] Youth sports team sponsorship ($200-500)
- [ ] Local charity sponsorship
- [ ] School events/fundraisers
- [ ] Local event sponsorships (farmers markets, festivals)

### Anti-Slop Link Rule
Every Core 30 page needs at least 1 quality local link pointing to it. This proves to Google a real entity has vouched for the content.

### Google News Press Releases
- [ ] Monthly press release (~$80 each) for brand mentions + links
- [ ] Helps both traditional SEO and AI visibility
- [ ] Target: 250 brand mentions across the web (magic number for AI systems to start recommending the business)

## Ongoing Monthly Fulfillment

### Weekly Tasks (VA-able)
- [ ] 1x GBP post (service spotlight, project photo, tip, seasonal content)
- [ ] Upload 2-3 new geotagged photos to GBP
- [ ] Check and respond to any new reviews
- [ ] Update client tracking sheet (Google Sheet the client has access to)

### Monthly Tasks
- [ ] Run LeadSnap rank map — compare to previous month
- [ ] Screenshot GBP insights (calls, views, direction requests)
- [ ] Send weekly summary of stats/KPIs to client (VA sends this)
- [ ] Monthly strategy call with client (30-45 mins, only thing YOU do personally)
- [ ] Identify border zones on rank map (position 4-5 dots) → create geographic content to push into top 3
- [ ] 1x Google News press release

### Quarterly Tasks
- [ ] Full title tag audit — update year in title tags (Gotch's "Phantom Technique")
- [ ] Content audit — identify thin pages, merge or improve
- [ ] Citation audit — check NAP consistency hasn't drifted
- [ ] Competitor check — have new competitors entered the Maps top 3?
